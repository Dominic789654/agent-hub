# Support

## Where to Ask for Help

- bug reports: open a GitHub issue with reproduction steps
- feature requests: open a GitHub issue with the user problem and proposed scope
- documentation or onboarding confusion: open a GitHub issue and call out the specific page or command
- security concerns: follow `SECURITY.md`

## What to Include

Please include:

- your platform and Python version
- the command you ran
- the project registry you used
- expected behavior
- actual behavior
- a minimal reproduction if possible

## Project Scope Reminder

`agent-hub` is currently a local-first OSS MVP for code-assistant task routing and visibility.

Good support requests are usually about:

- queue behavior
- dependency handling
- dispatcher behavior
- web or dashboard usability
- docs and examples

Requests that assume hosted SaaS behavior, multi-tenant auth, or private infrastructure are currently out of scope unless explicitly discussed first.

## Maintainer Response

Support is best-effort. There is no guaranteed response SLA.
